# chordcanvas

ローカルでコードダイアグラムを作成・編集・配置し、レイアウトを PDF 出力できるギター用コードダイアグラムエディタです。

## 現在の機能

- コードストックの `+` ボタンからコードビルダーモーダルを開き、current chord をストックへ追加できる
- 各レイアウト行の `+` ボタンから同じコードビルダーモーダルを開き、その行へコードブロックを追加できる
- 共通のコードビルダーモーダルでコード生成、押弦編集、コード情報確認をまとめて行える
- ヘッダーから表示 UI 言語を日本語 / English で切り替えられる
- レイアウト上の既存コードは編集モーダルを開いたときだけ更新できる
- current chord、レイアウト上のコード、コードストックに表示するコード名を手動で上書きできる
- 押弦マーカー内に `R`、`3`、`5`、`7`、`9` などのコードトーン degree を表示できる
- 複数の歌詞行にコードを配置できる
- レイアウト上の歌詞行をその場で直接編集でき、空白を含む表示位置を PDF 出力でも保てる
- 各コードごとに横オフセットと後続との間隔を調整でき、コードブロックを左右へドラッグして後続を押し出しながら配置できる
- よく使う押弦をコードストックに保存し、選択中の行やレイアウト追加モーダルから再追加できる
- project JSON の書き出し / 読み込みと、レイアウト PDF 出力に対応する

## セットアップ

1. `mise install` で固定された Node.js をインストールする
2. `npm install` で依存関係を入れる
3. `npm run dev` で開発サーバーを起動する

## 利用できるスクリプト

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run test`
- `npm run typecheck`
- `npm run format`
- `npm run format:check`
